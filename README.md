# ai_remoney — AI 事件驱动投资情报系统

**一句话**：预判金融大鳄的想法，在他们之前进场，或反制。

## 核心假设

中国 A 股市场本质是**投机市场**——几乎所有利润在当日兑现，获利即跑。因此：

> 价格由大资金的集体行为驱动，大资金的行为由信息驱动。
> 如果我们能拿到相同的信息，并推理出大资金会怎么做，就能抢跑或反制。

## 分析维度（8 维）

| 维度 | 监控内容 | 典型影响 |
|------|---------|---------|
| 1. 市场热度 | 前日/7日/本月交易量 | 水温计——过热警惕，冰点找机会 |
| 2. 国家政策 | 会议、政策方向、产业规划 | 政策驱动的板块轮动 |
| 3. 头部厂商 | 大厂动向、产业链变化 | 上下游联动效应 |
| 4. 消费端 | 使用者/消费者反响 | 需求端验证 |
| 5. 国际关系 | 国家间矛盾、制裁、合作 | 贵金属、稀有金属、能源 |
| 6. 供应链 | 航线堵塞、物流中断 | 石油、航运股 |
| 7. 突发灾害 | 自然灾害、事故 | 保险、建材、农产品 |
| 8. AI & 机器人 | 行业重大变化 | 制造业、半导体、软件 |

## 工作流

```
用户输入新闻 → 维度分类 → 影响分析 → 大资金行为推理 → 信号输出
```

## 核验等级

所有信息必须标注核验等级：

| 等级 | 含义 |
|------|------|
| V0 | 传闻/未证实 |
| V1 | 单一来源 |
| V2 | 多源交叉验证 |
| V3 | 官方/一手数据 |

**V0-V1 信息仅供标记，不作为进场依据。**
